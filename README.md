wol4t
===========

##What is it?

Twitter経由で任意のPCにWOLするプログラム。
ポート開放できなくてもRaspberryPiがあれば外部からWOLできるようになる。
telnetサーバーを有効にしておくとスリープにもできる。

推奨環境はlinux環境でrubyが必要です。
動作チェックはRaspberrypiのruby1.9.3p194です。


##How to Use

```
$ git clone https://github.com/flum1025/wol4t.git
$ ruby wol.rb &

```
